# BlockDrop

A modern Tetris game built with HTML5, CSS3, and JavaScript.

## Features

- Classic Tetris gameplay with all 7 tetromino shapes
- Ghost piece projection
- Next piece preview
- Particle effects on line clear
- Warning line indicator
- Combo system with INCREDIBLE effect
- Background music and sound effects
- Global leaderboard (Supabase)
- Sound toggle in pause menu

## How to Play

- **← →** Move piece
- **↓** Drop faster
- **↑ / Space** Rotate
- **P** Pause

## Scoring

- 1 Line = 75 points
- 2 Lines = 225 points
- 3 Lines = 475 points
- 4 Lines = 800 points
- Piece Lands = +10 points
- Combo Bonus = +25 points per consecutive clear
- INCREDIBLE! = +150 points (at 4+ combo)
